# AndyAI - 宝宝成长记录网站

## 1. 项目概述

**项目名称**: AndyAI
**项目类型**: 静态博客/时间轴网站
**核心功能**: 记录宝宝成长历程的时间轴式成长日记
**目标用户**: 父母、家人、朋友
**部署方式**: Vercel (GitHub 集成)
**风格**: 手绘温柔手账风

---

## 2. 视觉设计

### 2.1 配色方案

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景 | `#FDF8F3` | 奶油白，温暖纸质感 |
| 卡片 | `#FFFFFF` | 纯白 |
| 主色 | `#FFB6C1` | 浅粉色，温柔可爱 |
| 辅助色 | `#87CEEB` | 天空蓝，清新 |
| 点缀色 | `#FFD700` | 暖金色，记录重要时刻 |
| 文字主色 | `#5D4E37` | 暖棕色，手写感 |
| 文字次色 | `#8B7355` | 浅棕色 |
| 边框/分割 | `#E8DCC8` | 米色 |

### 2.2 字体选择

- **标题**: Google Fonts "ZCOOL XiaoWei" (站酷小薇)
- **正文**: Google Fonts "Noto Sans SC" (思源黑体)

### 2.3 手账风元素

- 撕边/贴纸效果
- 胶带装饰边框
- 便签纸样式卡片
- 手绘分割线
- 拍立得照片框
- 邮票/印章效果

## 3. 功能规划

### 3.1 页面结构

| 页面 | URL | 说明 |
|------|-----|------|
| 首页 | `/` | 时光时间轴，展示所有记录 |
| 分类页 | `/category/[cat]` | 按分类筛选 |
| 单页记录 | `/memory/[id]` | 每条记录的详情页 |
| 关于页 | `/about` | 宝宝介绍 |

### 3.2 里程碑分类

| 分类 | 标签 | 图标 |
|------|------|------|
| 满月/百天 | milestone | 🎀 |
| 生日 | birthday | 🎂 |
| 第一次 | first | ⭐ |
| 上学 | school | 📚 |
| 旅行 | travel | ✈️ |
| 日常 | daily | 🌸 |
| 节日 | festival | 🎉 |

### 3.3 核心功能

| 功能 | 优先级 | 说明 |
|------|--------|------|
| 时间轴展示 | P0 | 按时间顺序展示所有记录 |
| 照片展示 | P0 | 拍立得风格照片墙 |
| 里程碑标记 | P1 | 重要时刻特殊标记 |
| 分类筛选 | P1 | 按分类查看记录 |
| 搜索功能 | P2 | 关键词搜索记录 |
| 响应式设计 | P0 | 移动端适配 |

## 4. 技术方案

### 4.1 技术栈

| 技术 | 用途 |
|------|------|
| Next.js 15 (App Router) | 框架，静态生成 |
| Tailwind CSS | 样式 |
| Vercel | 部署 |
| GitHub | 代码托管 |

### 4.2 项目结构

```
AndyAI/
├── src/
│   ├── app/
│   │   ├── page.tsx           # 首页时间轴
│   │   ├── layout.tsx         # 布局
│   │   ├── globals.css         # 全局样式
│   │   ├── about/
│   │   │   └── page.tsx       # 关于页
│   │   └── memory/
│   │       └── [id]/
│   │           └── page.tsx    # 单条记录详情
│   ├── components/
│   │   ├── Timeline.tsx        # 时间轴组件
│   │   ├── MemoryCard.tsx     # 记忆卡片
│   │   ├── Header.tsx         # 头部
│   │   └── Footer.tsx         # 底部
│   └── data/
│       └── memories.ts         # 记忆数据
├── public/
│   └── photos/                # 照片目录
├── package.json
└── SPEC.md
```

### 4.3 静态生成

所有页面在构建时静态生成，JSON 数据存储在 `data/memories.ts`，无需数据库。
